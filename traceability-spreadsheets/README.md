# Traceability Spreadsheets

Included files:
- `ecif-traceability-workbook-template.xlsx`
- CSV extracts for quick import into project tools

Recommended usage:
1. Keep the workbook under version control with the modernization work artifacts.
2. Update Source_Target_Map and Business_Rules after each analysis cycle.
3. Require completion of Assumptions and Decision_Log before code generation approval.
4. Tie Test_Coverage rows back to rule IDs for auditability.
