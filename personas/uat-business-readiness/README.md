# Persona: UAT / Business Readiness

## Role Overview

UAT and Business Readiness teams verify the system meets business expectations before go-live. They bridge technical delivery and business operations, ensuring the system is not just built correctly but is ready for real users.

## What AI Can Help You With

- Generating business-language test scripts from user stories
- Creating go-live readiness checklists
- Drafting stakeholder communications for launches
- Generating training materials for end users
- Building sign-off documentation
- Creating cutover planning checklists

## Key Rule

UAT scripts must be written for **non-technical** users. No code, no HTTP status codes, no technical jargon. If a Card Operations Analyst can't follow the test script without help from an engineer, rewrite it.
