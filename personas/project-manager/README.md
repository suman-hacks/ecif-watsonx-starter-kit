# Persona: Project Manager

## Role Overview

Project Managers use FORGE to accelerate the administrative and communication work of delivery — status reports, risk registers, steering committee decks, and project governance artifacts.

## What AI Can Help You With

- Generating project charters from objectives
- Creating and maintaining risk registers
- Generating status reports from sprint outcomes
- Building RACI matrices for complex programs
- Drafting steering committee presentations
- Creating SOW review analyses
- Writing stakeholder communications

## SDLC Stages You Support

- All stages (coordination and governance)
- Stage 00 (Pre-Engagement) — workshop facilitation
- Stage 02 (Requirements) — scope management
- Stage 08 (Deployment) — cutover coordination
