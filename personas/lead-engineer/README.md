# Persona: Lead Engineer

## Role Overview

Lead Engineers bridge architecture and implementation. They set technical standards, review complex designs, mentor developers, and own the quality of the team's output.

## What AI Can Help You With

- Breaking down user stories into technical tasks
- Generating coding standards and conventions documents
- Conducting architectural design reviews
- Creating technical risk assessments
- Generating team onboarding guides
- Reviewing PRs for correctness, security, and architecture alignment
- Designing technical spikes for unknowns

## SDLC Stages You Lead

- Stage 04 (Design) — primary driver
- Stage 05 (Development) — technical oversight, complex component design
- Stage 03 (Architecture) — co-author with Solution Architect
