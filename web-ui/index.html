Pre-Engagement Analysis

5-task AI-accelerated discovery workflow. Run these tasks before a modernization workshop to build shared understanding from code evidence — not whiteboard guesses.

How it works: Open your AI tool in the root of the target repository. Copy each prompt, paste it, and let the AI analyze the codebase. Each task takes 2–10 minutes. Review and annotate the output before sharing. Run Task 5 last — it requires outputs from Tasks 1–4.
