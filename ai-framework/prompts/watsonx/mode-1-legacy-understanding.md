# Mode 1: Legacy Understanding

Use this mode before any design or code generation.

## Goal
Understand current behavior, dependencies, data access, and side effects.

## Prompt add-on
```text
Operate in Legacy Understanding mode.
Do not design the target solution yet.
Read the supplied source artifacts and produce:
- behavior summary
- entry points
- external dependencies
- data interactions
- error paths
- side effects
- ambiguities
```
