# Event-Driven Modernization for Payments Pack

## Purpose
Use this pack for event sourcing, CDC-driven propagation, Kafka integration, audit-first ingestion, replay/recovery, and consistency patterns in payment modernization.

## Use this pack for
- defining domain events
- designing asynchronous propagation
- handling downstream failures
- replaying failed operations
- building audit/reprocessing patterns
- creating read models and reconciliation flows
