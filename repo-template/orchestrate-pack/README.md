# watsonx Orchestrate Workflow Pack

This pack is designed so a team can externalize the ECIF operating model into a workflow with explicit handoffs and review points.

## Workflow stages
1. Source intake and artifact indexing
2. Legacy understanding
3. Business rule extraction
4. Target mapping
5. Code generation
6. Review and hardening
7. Traceability and approval package generation

## Included assets
- `workflow-definition.yaml`
- `agents/*.md`
- `playbooks/*.md`
- `handoff-contracts/*.json`
- `prompts/*.md`
