# FORGE — AI-Powered Framework for Orchestrated, Reliable Guided Engineering

> **The Enterprise AI Engineering Framework**  
> From Legacy to Modern. From Requirement to Release.

---

## What is FORGE?

FORGE is a vendor-neutral, enterprise-grade framework that brings structure, governance, and repeatable process to AI-assisted software engineering. It provides:

- **Structured prompts** for every stage of the SDLC and PDLC
- **Role-specific guidance** for every engineering persona — from PM to DevSecOps
- **Tool configurations** for GitHub Copilot, Claude Code, watsonx Code Assist for Z, IBM BAM, Cursor, and AWS Q Developer
- **Governance artifacts** — constitution, guardrails, handoff contracts, audit trails
- **Knowledge packs** — domain and technical context loaded on demand
- **Project contexts** — pre-configured for greenfield, brownfield, mainframe modernization, and cloud migration

FORGE does not replace your engineers or your tools. It makes both dramatically more effective by providing the structure that prevents AI from guessing, hallucinating, and short-cutting — and channels it toward precise, traceable, production-quality output.

---

## Core Design Principle

```
Understand → Extract → Design → Build → Validate → Operate
```

No stage can be skipped. No code is generated without analysis. No architecture is implemented without review. The constitution enforces this at every step.
